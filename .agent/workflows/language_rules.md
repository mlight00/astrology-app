# 언어 설정 (Language Preference)

사용자의 요청에 따라 앞으로의 모든 대화, 코드 주석, 결과물, 웹사이트의 UI 텍스트는 **한국어(Korean)**를 기본으로 사용합니다.


## 규칙

1. **사용자 응답**: AI의 모든 답변은 한국어로 작성한다.
2. **코드 및 주석**: 변수명은 영어를 유지하되, 주석과 사용자에게 보여지는 UI 텍스트는 한국어로 번역한다.
3. **웹사이트 콘텐츠**: 웹 애플리케이션의 모든 레이블, 메시지, 데이터는 한국어로 표시한다.
