# 🔮 사주팔자 전문 상담 (Astrology Consultation)

전문적인 사주팔자 상담을 위한 웹 애플리케이션입니다. 
사용자의 생년월일시 정보를 바탕으로 오행(五行)과 대운(大運)을 분석하여 시각적인 대시보드와 실천적인 개운법을 제공합니다.

## ✨ 주요 기능

*   **전문적인 디자인**: 몰입감을 주는 다크 모드와 글래스모피즘(Glassmorphism) UI.
*   **오행 시각화**: 목(木), 화(火), 토(土), 금(金), 수(水) 오행 분포를 아름다운 차트로 시각화.
*   **운명의 시그니처**: 사주를 한 줄로 요약하는 '운명의 한 줄' 기능.
*   **대운 흐름 분석**: 인생의 흐름을 계절(봄, 여름, 가을, 겨울)에 비유하여 시각적으로 표현.
*   **맞춤형 전략**: 상담 주제(재물, 직업, 애정, 건강)에 따른 구체적인 행동 지침 제공.

## 🛠 기술 스택

*   **Core**: React, Vite
*   **Styling**: Vanilla CSS (Variables, Glassmorphism), Lucide React (Icons)
*   **Animation**: Framer Motion
*   **Deployment**: Vercel

## 🚀 시작하기

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

### 빌드

```bash
npm run build
```

## 🌐 배포 (Deployment)

이 프로젝트는 [Vercel](https://vercel.com)에 최적화되어 있습니다.

1. GitHub 리포지토리를 Vercel에 임포트합니다.
2. `Build Command`는 `npm run build`, `Output Directory`는 `dist`로 설정합니다(Vite 기본값).
3. **Deploy** 버튼을 누르면 배포가 완료됩니다.
